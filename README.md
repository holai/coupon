# coupon

接入 淘宝联盟 做的小 dome
目前实现

- 商品列表
- 搜索
- 红包

使用 Getx 框架，作为工作外的练习

后端使用 node.js ， 目录 node   启动命令 node www

部署配置

后端配置 需要修改 apiClient 中的 appkey 与 appsecret 修改为自己的
前端需要修改 lib/config/config 下的请求地址
